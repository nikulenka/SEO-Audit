# Отчет по техническому фундаменту (AI Readiness) — CTT.BY

## 1. Проверка robots.txt
- **Статус:** ⚠️ Требует улучшения.
- **Анализ:** В файле отсутствуют явные инструкции для современных ИИ-краулеров (`GPTBot`, `ClaudeBot`, `PerplexityBot`, `OAI-SearchBot`). Используются стандартные правила WordPress.
- **Рекомендация:** Добавить блок разрешений для ИИ-ботов, чтобы ускорить индексацию новых материалов по ЭДО и маркировке.

## 2. Анализ SSR (Server-Side Rendering)
- **Статус:** ✅ Отлично.
- **Анализ:** Сравнение ответов для обычного браузера и Googlebot показало идентичный размер (360 КБ) и структуру. Весь критический контент (меню, заголовки, описание услуг) доступен в исходном HTML без выполнения JavaScript.
- **Результат:** ИИ-движки увидят полный текст страницы сразу.

## 3. Проверка llms.txt
- **Статус:** ❌ Отсутствует (404).
- **Анализ:** Файл по адресу `https://ctt.by/llms.txt` не найден. Это стандартный способ "объяснить" структуру сайта ИИ-моделям.
- **Рекомендация:** Создать файл `llms.txt` в корне сайта с кратким описанием ключевых сервисов (EDI, Маркировка, Интеграция).

## 4. Аудит микроразметки (Tech)
- **Статус:** ✅ Базовый уровень пройден.
- **Анализ:** На главной странице присутствует разметка `WebSite` и `Organization` (через Yoast SEO).
- **Замечания:**
  - `Organization` содержит `sameAs` ссылки на Facebook, Instagram, LinkedIn, Telegram. Это хорошо для связки сущностей.
  - Рекомендуется расширить разметку `Organization` данными о контактных лицах и конкретных услугах в формате JSON-LD.

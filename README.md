# SEO-Audit: AI-Powered Technical SEO Analyzer

**SEO-Audit** — это профессиональный веб-инструмент для глубокого технического анализа сайтов и автоматического подбора SEO-оптимизаций с использованием возможностей искусственного интеллекта (Gemini 2.5 Flash).

Приложение сканирует структуру сайта (краулинг), собирает метатеги, анализирует контент и выдает подробный дашборд со здоровьем веб-ресурса, ключевыми словами, рекомендациями по улучшению и списком главных конкурентов. Открывается как стильный SPA (Single Page Application) с темной темой (Dark Mode) и Glassmorphism-дизайном, а также умеет выгружать отчет в красивый и читабельный PDF-файл.

---

## 🌟 Основные возможности

1. **Глубокий краулинг сайта:**
   * Сбор H1-H6 заголовков, Meta title, Meta description.
   * Выявление сломанных ссылок (404), дубликатов страниц и отсутствующих тегов.
   * Оценка общего "здоровья" (Health Score) оптимизации сайта.

2. **Искусственный интеллект (Google Gemini):**
   * Автоматический подбор **Top-15 ключевых слов** (с оценкой объема поиска и конкуренции).
   * Определение **Top-5 конкурентов** ресурса в выбранной нише.
   * Анализ контента с предложениями по созданию таблиц для забора сниппетов (AI Overviews) в поисковых системах.

3. **Современный UI/UX интерфейс:**
   * Адаптивный и "живой" дизайн, стекло-эффекты и плавные анимации.
   * Динамическое отображение прогресса сканирования (Progress Bar с детальным статусом).

4. **Генерация отчетов (Print-to-PDF):**
   * Экспорт подробного аудита в высококачественный PDF (с сохранением разметки и выделяемым текстом) при помощи нативных движков печати (без багов сторонних JS-библиотек).

## 🛠 Технологический стек

* **Backend:** Python + Flask, threading для асинхронного краулинга.
* **Сбор данных:** `requests`, `BeautifulSoup4` (bs4).
* **AI Интеграция:** Официальный Python SDK `google-genai` (модель `gemini-2.5-flash`).
* **Frontend:** Vanilla HTML5, CSS3, JavaScript (полноценный SPA-клиент).
* **Сервер для продакшена:** Gunicorn.
* **Защита тайн:** `python-dotenv` для работы с переменными окружения (.env).

## 🚀 Как запустить локально (для разработки)

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/nikulenka/SEO-Audit.git
   cd SEO-Audit
   ```

2. **Настройте виртуальное окружение (рекомендуется):**
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   ```

3. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

4. **Настройте ключи API:**
   Создайте файл `.env` в корне проекта на основе `.env.example`:
   ```bash
   cp .env.example .env
   ```
   Откройте `.env` и вставьте ваш собственный ключ от Google Gemini Studio:
   `GEMINI_API_KEY=ваш_секретный_ключ`

5. **Запустите приложение:**
   ```bash
   python app.py
   ```
   Откройте браузер по адресу: `http://localhost:5000`

## 🌍 Как запустить в облаке (Deploy на Render.com)

Этот сервис полностью готов для развертывания через GitHub напрямую в [Render](https://render.com/).

1. Зарегистрируйтесь на Render и создайте новый **Web Service**.
2. Подключите ваш GitHub репозиторий с проектом.
3. Укажите следующие параметры для сборки:
   * **Language**: Python
   * **Build Command**: `pip install -r requirements.txt`
   * **Start Command**: `gunicorn app:app`
4. В разделе **Environment Variables** (переменные окружения) добавьте:
   * Key: `GEMINI_API_KEY`
   * Value: `<ваш_ключ_от_Google_AI>`
5. Создайте сервис, и через пару минут он запустится по публичному адресу.

## 📝 Разработчикам

Структура проекта:
- `app.py`: Главный серверный роутер + подключение AI.
- `crawler_engine.py`: Ядро многопоточного сканера сайтов на Python.
- `requirements.txt`: Список всех необходимых pip-пакетов для деплоя.
- `/static`: Стили приложения (`style.css`) и вся интерфейсная логика (`app.js`).
- `/templates`: Единственный HTML-шаблон для рендеринга `index.html`.
